# Agent Review Rule

## Mindset

Khi nguoi dung yeu cau review, Agent phai uu tien:

- bug
- regression risk
- contract mismatch
- missing validation
- missing test

Khong bien review thanh tom tat thay doi.

## Format nen tra

1. Findings truoc.
2. Sap xep theo muc do nghiem trong.
3. Moi finding nen noi:
   - van de
   - tac dong
   - file/vi tri lien quan
   - huong sua neu co
4. Neu khong thay finding ro, noi ro dieu do va neu residual risk.
