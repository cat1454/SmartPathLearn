# Agent Done Check Rule

## Truoc khi chot task

Agent phai tu check:

1. Da sua root cause hay chua.
2. Da verify bang cach tot nhat co the hay chua.
3. Co file nao bi doi ma khong lien quan khong.
4. Da note thay doi vao `.local-agent-rules/CHANGELOG.md` hay chua.
5. Co assumption nao can noi ro khong.
6. Co edge case moi nao vua mo ra khong.
7. Neu khong verify duoc, da noi thang ly do chua.
